## 简介

BillMii 是一款专为 Android 设备设计的票据和发票管理应用，专注于本地 OCR 识别、智能分类归档和财务报销管理。通过 BillMii，您可以轻松管理各种票据、发票和凭证，实现智能化的财务报销流程。

### 主要特点

- **本地 OCR 识别**：基于 PaddleOCR PP-OCRv4 Mobile，支持离线识别
- **智能分类**：自动将票据分类到不同的类别中
- **报销管理**：创建和管理报销单，支持审批流程
- **数据导出**：支持导出为 Excel 和 PDF 格式
- **数据安全**：支持生物识别认证和数据加密
- **语音输入**：支持语音输入描述信息
- **扫码功能**：支持扫描二维码快速录入票据信息

## 快速开始

### 安装应用

1. 从应用商店或官方网站下载 BillMii APK 文件
2. 在 Android 设备上安装应用
3. 首次启动时，授予必要的权限（相机、存储、麦克风等）
4. 设置生物识别认证（可选）

### 首次使用

1. **添加票据**：点击首页的"+"按钮，选择"拍照"或"从相册选择"
2. **OCR 识别**：系统会自动识别票据内容
3. **编辑信息**：检查识别结果，手动修正错误信息
4. **保存票据**：点击"保存"按钮完成添加

## 票据管理

### 添加票据

**方式一：拍照**

1. 点击首页的"+"按钮
2. 选择"拍照"
3. 对准票据拍照
4. 系统自动进行 OCR 识别
5. 编辑识别结果
6. 保存票据

**方式二：从相册选择**

1. 点击首页的"+"按钮
2. 选择"从相册选择"
3. 选择要添加的票据图片
4. 系统自动进行 OCR 识别
5. 编辑识别结果
6. 保存票据

**方式三：扫码录入**

1. 点击首页的扫码按钮
2. 对准票据上的二维码扫描
3. 系统自动解析二维码信息
4. 编辑信息
5. 保存票据

### 编辑票据

1. 在票据列表中点击要编辑的票据
2. 点击"编辑"按钮
3. 修改需要更改的信息
4. 点击"保存"按钮

### 删除票据

**单个删除**

1. 在票据详情页点击"删除"按钮
2. 确认删除操作

**批量删除**

1. 在票据列表中长按票据进入选择模式
2. 选择要删除的票据
3. 点击"删除"按钮
4. 确认删除操作

### 搜索票据

1. 点击搜索按钮
2. 输入搜索关键词（标题、商户名称、备注等）
3. 或使用语音输入按钮进行语音搜索
4. 查看搜索结果

### 筛选票据

1. 点击筛选按钮
2. 设置筛选条件：
   - 日期范围
   - 类别
   - 状态
   - 金额范围
3. 点击"应用"按钮
4. 查看筛选结果

## OCR 识别

### 自动识别

BillMii 会在添加票据时自动进行 OCR 识别，提取以下信息：

- 票据标题
- 商户名称
- 金额
- 日期
- 商品/服务详情
- 其他相关信息

### 手动识别

如果自动识别失败或需要重新识别：

1. 打开票据详情页
2. 点击"重新识别"按钮
3. 系统会重新进行 OCR 识别

### OCR 模板设置

如果某些票据识别效果不佳，可以设置 OCR 模板：

1. 进入设置 → OCR 模板
2. 点击"添加模板"
3. 上传示例票据图片
4. 标记需要识别的区域
5. 设置识别规则
6. 保存模板

## 智能分类

### 自动分类

BillMii 会根据票据内容和设置的分类规则自动将票据分类到相应的类别中。

### 分类规则设置

1. 进入设置 → 分类规则
2. 点击"添加规则"
3. 设置规则名称
4. 设置匹配条件（关键词、金额范围、商户等）
5. 设置目标类别
6. 保存规则

### 手动分类

1. 打开票据详情页
2. 点击"分类"按钮
3. 选择目标类别
4. 确认分类

## 报销管理

### 创建报销单

1. 进入报销页面
2. 点击"创建报销单"按钮
3. 填写报销单信息：
   - 报销标题
   - 申请人
   - 部门
   - 报销类别
   - 费用类型
   - 项目代码（可选）
   - 描述（支持语音输入）
4. 点击"添加票据"按钮
5. 选择要添加的票据
6. 点击"保存草稿"或"提交"

### 编辑报销单

1. 打开报销单详情页
2. 点击"编辑"按钮
3. 修改需要更改的信息
4. 点击"保存"按钮

### 提交报销单

1. 打开报销单详情页
2. 点击"提交"按钮
3. 确认提交

### 查看报销状态

报销单有以下状态：

- **草稿**：尚未提交的报销单
- **已提交**：已提交等待审批
- **已通过**：审批通过
- **已拒绝**：审批被拒绝
- **已支付**：已完成支付
- **已取消**：已取消的报销单

### 报销审批

1. 进入审批页面
2. 查看待审批的报销单
3. 打开报销单详情
4. 点击"通过"或"拒绝"按钮
5. 添加审批意见（可选）
6. 确认操作

## 数据导出

### 导出为 Excel

1. 进入票据列表或报销单列表
2. 点击"导出"按钮
3. 选择"导出为 Excel"
4. 选择导出模板（可选）
5. 选择导出范围（全部或选中项）
6. 点击"导出"按钮
7. 选择保存位置

### 导出为 PDF

1. 进入票据列表或报销单列表
2. 点击"导出"按钮
3. 选择"导出为 PDF"
4. 选择导出模板（可选）
5. 选择导出范围（全部或选中项）
6. 点击"导出"按钮
7. 选择保存位置

### 自定义导出模板

1. 进入设置 → 导出模板
2. 点击"添加模板"
3. 设置模板名称
4. 选择导出格式（Excel 或 PDF）
5. 配置导出字段
6. 设置样式（PDF）
7. 保存模板

## 数据备份与恢复

### 手动备份

1. 进入设置 → 数据管理
2. 点击"立即备份"
3. 选择备份位置
4. 设置备份密码（可选）
5. 点击"备份"按钮
6. 等待备份完成

### 自动备份

1. 进入设置 → 数据管理
2. 开启"自动备份"
3. 设置备份间隔（每天、每周、每月）
4. 设置备份位置
5. 设置备份密码（可选）
6. 保存设置

### 恢复数据

1. 进入设置 → 数据管理
2. 点击"恢复数据"
3. 选择备份文件
4. 输入备份密码（如果设置了）
5. 确认恢复操作
6. 等待恢复完成

**注意**：恢复数据会覆盖当前数据，请谨慎操作。

### 备份文件管理

1. 进入设置 → 数据管理
2. 点击"备份文件管理"
3. 查看所有备份文件
4. 可以删除不需要的备份文件
5. 可以查看备份文件详情

## 常见问题

### Q: OCR 识别不准确怎么办？

A: 可以尝试以下方法：
- 确保拍摄时光线充足
- 保持票据平整
- 避免反光和阴影
- 使用 OCR 模板功能

### Q: 如何提高分类准确率？

A: 可以：
- 设置详细的分类规则
- 手动修正分类错误
- 定期检查和更新规则

### Q: 备份数据在哪里？

A: 备份数据默认保存在：
- 内部存储：`/Android/data/com.billmii.android/files/backup/`
- 外部存储：如果设置了外部存储路径

### Q: 如何导出所有票据？

A:
1. 进入票据列表
2. 点击"导出"按钮
3. 选择导出格式
4. 选择"全部"导出范围
5. 点击"导出"按钮

### Q: 报销单被拒绝了怎么办？

A:
1. 查看拒绝原因
2. 根据拒绝原因修改报销单
3. 重新提交

### Q: 如何删除所有数据？

A:
1. 进入设置 → 数据管理
2. 点击"清除所有数据"
3. 确认删除操作

**警告**：此操作不可逆，请谨慎操作。

### Q: 应用无法启动怎么办？

A: 可以尝试：
- 重启应用
- 清除应用缓存
- 重新安装应用
- 联系技术支持
